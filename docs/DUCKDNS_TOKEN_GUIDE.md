# Как получить токен DuckDNS

## Шаг 1: Откройте сайт DuckDNS

Перейдите на https://www.duckdns.org

## Шаг 2: Войдите в аккаунт

На главной странице вверху есть кнопки для входа через:
- Twitter
- GitHub  
- Google
- Reddit

Выберите тот сервис, через который вы регистрировались.

## Шаг 3: Найдите токен

После входа в аккаунт:

1. **Токен находится в самом верху страницы** в желтом блоке
2. Он выглядит примерно так: `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` (формат UUID)
3. Рядом с токеном есть текст: "your token is:"

**Пример расположения:**
```
┌─────────────────────────────────────────────────┐
│ your token is: 12345678-1234-1234-1234-123456789012 │
└─────────────────────────────────────────────────┘
```

## Шаг 4: Скопируйте токен

Просто выделите и скопируйте строку токена (без текста "your token is:").

## Шаг 5: Используйте токен

После получения токена выполните:

```bash
# Замените YOUR_TOKEN на скопированный токен
curl "https://www.duckdns.org/update?domains=makishvili&token=YOUR_TOKEN&ip=89.169.189.202"
```

**Ожидаемый ответ:** `OK`

Если получили `KO` - токен неверный или домен не принадлежит вашему аккаунту.

## Альтернатива: Автоматическое определение IP

Если хотите, чтобы DuckDNS автоматически определил ваш текущий IP:

```bash
curl "https://www.duckdns.org/update?domains=makishvili&token=YOUR_TOKEN&ip="
```

(обратите внимание на пустой параметр `ip=`)

## Проверка после обновления

Подождите 2-3 минуты и проверьте:

```bash
# Проверка через Cloudflare DNS
dig @1.1.1.1 makishvili.duckdns.org +short

# Должен вернуть: 89.169.189.202
```

## Если не можете войти

Если вы не помните, через какой сервис регистрировались:

1. Попробуйте все варианты входа (Twitter, GitHub, Google, Reddit)
2. Проверьте email - возможно, есть письма от DuckDNS
3. Если не получается - создайте новый домен на DuckDNS и обновите конфигурацию бота

## Безопасность токена

⚠️ **Важно:** Токен DuckDNS - это секретный ключ. Не публикуйте его в открытых репозиториях или чатах.

Если токен скомпрометирован:
1. Зайдите на DuckDNS
2. Нажмите "regenerate token" (если такая опция есть)
3. Или удалите и создайте домен заново
